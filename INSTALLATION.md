## 📋 חלק 1: הגדרת Google Sheets (10 דקות)

### שלב 1: יצירת הגיליון

1. היכנסו ל-[Google Sheets](https://sheets.google.com)
2. לחצו על **"+"** (גיליון חדש)
3. שנו את השם ל: **"ציוני משחקי שוק ההון"**

### שלב 2: הגדרת עמודות

בשורה הראשונה, הזינו את הכותרות הבאות:

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| שם תלמיד | שם משחק | רמת קושי | ציון | תאריך ושעה | זמן משחק |

**כך זה צריך להיראות:**
```
A1: שם תלמיד
B1: שם משחק
C1: רמת קושי
D1: ציון
E1: תאריך ושעה
F1: זמן משחק
```

### שלב 3: יצירת Google Apps Script

1. בתפריט העליון: **Extensions → Apps Script**
2. **מחקו** את כל הקוד הקיים
3. **העתיקו והדביקו** את הקוד הבא:

```javascript
function doPost(e) {
  try {
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    const data = JSON.parse(e.postData.contents);
    
    sheet.appendRow([
      data.studentName,
      data.gameName,
      data.level,
      data.score,
      data.timestamp,
      data.duration
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({
      status: 'success',
      message: 'הציון נשמר בהצלחה!'
    })).setMimeType(ContentService.MimeType.JSON);
    
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({
      status: 'error',
      message: error.toString()
    })).setMimeType(ContentService.MimeType.JSON);
  }
}
```

4. לחצו על **💾 Save** (שמרו בשם "updateSheet")

### שלב 4: פרסום הסקריפט

1. לחצו על **Deploy** (למעלה מימין)
2. בחרו **New deployment**
3. לחצו על ⚙️ (גלגל שיניים) → **Web app**
4. מלאו את הפרטים:
   - **Description**: "API לעדכון ציונים"
   - **Execute as**: **Me** (הכתובת שלכם)
   - **Who has access**: **Anyone** ⚠️ **זה חשוב מאוד!**
5. לחצו **Deploy**
6. אישרו הרשאות (אם מתבקש):
   - לחצו **Review permissions**
   - בחרו את החשבון שלכם
   - לחצו **Advanced** → **Go to... (unsafe)**
   - לחצו **Allow**
7. **📋 העתיקו את ה-URL** (שמרו אותו!)
   - ה-URL נראה כך: `https://script.google.com/macros/s/...`

### שלב 5: הוספת ה-URL לקוד

1. פתחו את הקובץ **`js/google-sheets.js`** בעורך טקסט
2. מצאו את השורה:
   ```javascript
   const GOOGLE_SHEETS_URL = 'YOUR_WEB_APP_URL_HERE';
   ```
3. החליפו `YOUR_WEB_APP_URL_HERE` ב-URL שהעתקתם (בתוך המרכאות!)
4. לדוגמה:
   ```javascript
   const GOOGLE_SHEETS_URL = 'https://script.google.com/macros/s/ABC123.../exec';
   ```
5. **שמרו את הקובץ**

### שלב 1: כניסה ל-Base44

## ❌ פתרון בעיות נפוצות

### 🔴 בעיה 1: "שגיאה בשליחת הציון"

**פתרונות:**

**א. בדקו את ה-URL:**
- ודאו שה-URL בקובץ `google-sheets.js` נכון
- ודאו שאין רווחים בתחילת או בסוף ה-URL
- ודאו שה-URL מתחיל ב-`https://`

**ב. בדקו הרשאות:**
- חזרו ל-Google Apps Script
- **Deploy → Manage deployments**
- ודאו ש-"Who has access" הוא **"Anyone"** ולא "Only me"
- אם לא, עשו **New deployment** שוב

**ג. פרסמו מחדש:**
1. חזרו ל-Apps Script
2. לחצו **Deploy → New deployment**
3. קבלו URL חדש
4. החליפו בקובץ `google-sheets.js`
5. העלו מחדש ל-Base44

### 🔴 בעיה 2: המשחקים לא נטענים

**פתרונות:**

**א. בדקו את הקבצים:**
- ודאו שכל הקבצים הועלו ל-Base44
- ודאו שמבנה התיקיות נכון:
  ```
  css/
  js/
  assets/
  index.html
  ```

**ב. בדקו שגיאות:**
1. לחצו **F12** בדפדפן
2. לחצו על **Console**
3. חפשו שגיאות אדומות
4. צלמו מסך ושלחו תמיכה

**ג. נקו Cache:**
1. לחצו **Ctrl + Shift + Delete**
2. בחרו "Cached images and files"
3. לחצו **Clear data**
4. רעננו את הדף

### 🔴 בעיה 3: הדשבורד לא נראה טוב במובייל

**פתרונות:**

- ודאו שהקובץ `css/responsive.css` הועלה
- נקו Cache במכשיר הנייד
- נסו לרענן את הדף (משיכה למטה)
