# 🎮 אתר משחקים חינוכיים - שוק ההון לחטיבות ביניים

## 📖 תיאור הפרויקט

פלטפורמת למידה אינטראקטיבית לחינוך פיננסי המיועדת לתלמידי כיתות ח'-ט'. האתר כולל 5 משחקים חינוכיים מדורגים בשלוש רמות קושי, מערכת מעקב למורים, ואינטגרציה עם Google Sheets.

## ✨ תכונות עיקריות

✅ **דשבורד מרכזי** - כל המשחקים במקום אחד  
✅ **5 משחקים אינטראקטיביים מלאים**  
✅ **3 רמות קושי** לכל משחק (קל, בינוני, קשה)  
✅ **מעקב אוטומטי** של ציונים ב-Google Sheets  
✅ **דשבורד מיוחד למורים** עם סטטיסטיקות וגרפים  
✅ **עיצוב רספונסיבי** - עובד במחשב, טאבלט ומובייל  
✅ **תמיכה מלאה בעברית** - ממשק וממשק משתמש בעברית  
✅ **נגיש** - עומד בתקן WCAG 2.1

## 🎯 המשחקים

### 1. סימולטור בורסה 📈
קבלו תקציב וירטואלי ובנו תיק השקעות! עקבו אחר מחירי מניות בזמן אמת וקבלו החלטות חכמות.

**מיומנויות:** אחוזים, גרפים, חישוב רווח והפסד

### 2. קורא המגמות 📊
נתחו גרפים פיננסיים אמיתיים! זהו מגמות, חשבו אחוזי שינוי והשוו ביצועים.

**מיומנויות:** ניתוח נתונים, קריאת גרפים, השוואה

### 3. חידות המדד 🧩
פתרו חידות לוגיות-מתמטיות! חשבו ריבית דריבית וחישובים פיננסיים מורכבים.

**מיומנויות:** ריבית דריבית, חשיבה לוגית, מתמטיקה פיננסית

### 4. מלחמת התיקים 🏆
תחרות קבוצתית מרתקת! בנו תיק השקעות ביחד והתמודדו עם אירועי שוק.

**מיומנויות:** שיתוף פעולה, החלטות מהירות, אסטרטגיה

### 5. דילמות המשקיע 💡
התמודדו עם דילמות השקעה אמיתיות! קבלו החלטות מושכלות ולמדו על סיכון ותשואה.

**מיומנויות:** קבלת החלטות, ניהול סיכונים, חשיבה אסטרטגית

## 💻 טכנולוגיות

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Charts:** Chart.js 4.x
- **Data Storage:** LocalStorage + Google Sheets API
- **Icons:** Font Awesome 6
- **Fonts:** Heebo, Assistant

## 📦 התקנה מהירה

### שלב 1: הורדה
הורידו את כל הקבצים לתיקייה אחת

### שלב 2: הגדרת Google Sheets
1. צרו גיליון חדש ב-Google Sheets
2. עקבו אחר ההוראות ב-`INSTALLATION.md`
3. העתיקו את ה-URL לקובץ `js/google-sheets.js`

### שלב 3: העלאה ל-Base44
1. היכנסו ל-[Base44.com](https://base44.com)
2. New Project → Upload Files
3. העלו את כל התיקייה
4. Publish → קבלו לינק

### שלב 4: בדיקה
פתחו את הלינק ונסו לשחק משחק!

## 🐛 פתרון בעיות

### בעיה: "שגיאה בשליחת הציון"
**פתרון:**
- ודאו שה-URL בקובץ `google-sheets.js` נכון
- ודאו שבסקריפט Google בחרתם "Anyone" ולא "Only me"
- נסו לפרסם מחדש את הסקריפט

### בעיה: המשחקים לא נטענים
**פתרון:**
- ודאו שכל הקבצים הועלו
- בדקו Console (F12) לשגיאות
- נקו Cache (Ctrl + Shift + Del)

### בעיה: הדשבורד לא נראה טוב במובייל
**פתרון:**
- ודאו שהקובץ `css/responsive.css` הועלה
- נקו Cache בדפדפן
