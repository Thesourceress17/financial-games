## 🔗 חיבור ל-Google Sheets (10 דקות)

**למה צריך את זה?**  
כדי שכל הציונים של התלמידים יישמרו בגיליון Google אחד שאת תוכלי לראות.

### שלב 1: צרי גיליון
1. לכי ל-[Google Sheets](https://sheets.google.com)
2. צרי גיליון חדש
3. בשורה ראשונה, כתבי:
   ```
   A1: שם תלמיד
   B1: שם משחק
   C1: רמת קושי
   D1: ציון
   E1: תאריך ושעה
   F1: זמן משחק
   ```

### שלב 2: העתיקי קוד
1. בגיליון: **Extensions → Apps Script**
2. מחקי הכל
3. פתחי את הקובץ `docs/google-apps-script.js`
4. **העתיקי את כל הקוד** והדביקי בגיליון
5. לחצי **Save** (שמי בשם "updateSheet")

### שלב 3: פרסמי
1. לחצי **Deploy → New deployment**
2. בחרי **Web app**
3. **חשוב:** ב-"Who has access" בחרי **"Anyone"**
4. לחצי **Deploy**
5. **העתיקי את ה-URL!** (זה חשוב!)

### שלב 4: הוסיפי ל-Code
1. פתחי את הקובץ `js/google-sheets.js`
2. מצאי את השורה:
   ```javascript
   const GOOGLE_SHEETS_URL = 'YOUR_WEB_APP_URL_HERE';
   ```
3. **החליפי** את `YOUR_WEB_APP_URL_HERE` ב-URL שהעתקת
4. **שמרי!**
5. **העלי מחדש ל-Base44** (גררי את הקובץ המעודכן)

**זהו! עכשיו הציונים יישלחו אוטומטית! 🎊**

## ❓ שאלות נפוצות

### ש: האתר לא נראה טוב במובייל?
**ת:** האתר **רספונסיבי לחלוטין**! אם זה לא נראה טוב, נסי:
- לנקות Cache (Ctrl+Shift+Del)
- לרענן את הדף
- לבדוק שכל הקבצים הועלו

### ש: איך אני משנה את הסיסמה של המורה?
**ת:** פתחי את `js/app.js`, חפשי:
```javascript
if (password === 'teacher123')
```
שני ל:
```javascript
if (password === 'הסיסמה_שלך')
```
ושמרי, העלי מחדש.

### ש: הציונים לא נשלחים לגוגל?
**ת:** בדקי:
1. ה-URL ב-`google-sheets.js` נכון?
2. פרסמת את הסקריפט עם "Anyone"?
3. העלית את הקובץ המעודכן ל-Base44?

### ש: אני רוצה לשנות צבעים/עיצוב?
**ת:** כל הצבעים ב-`css/main.css` בתחילת הקובץ (`:root`).

### ש: איך אני מוסיפה מניות נוספות?
**ת:** ב-`js/games/game1-stock-simulator.js`, חפשי `STOCK_NAMES` והוסיפי.
